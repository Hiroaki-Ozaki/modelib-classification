#### 訓練時

- 訓練は `python train.py`にて実行可能です。ただし、初期設定を行った上で実行してください。

- 初期設定ファイルはconfig.iniにて行ってください。
  - model_nameには以下が利用できます。
    - VGG16
    - VGG19
    - InceptionV3
    - DenseNet

- 作ったモデルとログはcheckpoints内に格納されます。



#### テスト時

- `python test.py`にて実行してください。
